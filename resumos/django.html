===== Introdução =====


Instalação:

  • Possuir Python 3 full e Pycharm instalados. Criar novo projeto Python, com environment Virtualenv (Atentar-se a localização do mesmo)
  • Instalar Pip: sudo apt install python3-pip
  • Instalar Django: pip install django
  • Criar arquivo de lista de dependências: pip freeze > requirements.txt
