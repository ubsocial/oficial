Instalar Eclipse no Debian

Instalação e atalho do Eclipse Java no Debian
Voltar

Passo a passo:



  1. Pré-requisito: OpenJDK (sudo apt install default-jdk)
  2. Baixar, na pasta Downloads, Eclipse Java oficial (Arquivo .tar.gz): Acesse
  3. Extrair arquivo e mover pasta extraída ('eclipse') para a pasta Home
  4. Criar atalho: sudo nano /usr/share/applications/eclipse-java.desktop 
    
Inserir o seguinte conteúdo no arquivo. Ctrl+k para fechar, e salvá-lo
[Desktop Entry]
Encoding=UTF-8
Exec=/home/SEUUSUARIO/eclipse/eclipse
Icon=/home/SEUUSUARIO/eclipse/icon.xpm
Type=Application
Terminal=false
Comment=Eclipse Integrated Development Environment
Name=Eclipse
GenericName=Eclipse
StartupNotify=false
Categories=Development;IDE;Java;
