Instalar IntelliJ no Debian

Instalação do IntellJ Community no Debian
Voltar

Passo a passo:



  1. Pré-requisito: OpenJDK (sudo apt install default-jdk)
  2. Baixar, na pasta Downloads, Intellij Community oficial (Arquivo .tar.gz): Acesse
  3. cd Downloads && tar xvzf ideaIC-*.tar.gz && mv idea*/ idea-community
  4. sudo mv idea-community /opt
  5. sudo ln -s /opt/idea-community/bin/idea.sh /usr/bin/idea
  6. Criar atalho: sudo nano /usr/share/applications/jetbrains-idea-ce.desktop 
    
Inserir o seguinte conteúdo no arquivo. Ctrl+k para fechar, e salvá-lo
[Desktop Entry]
Version=1.0
Type=Application
Name=IntelliJ Community Edition
Icon=/opt/idea-community/bin/idea.png
Exec="/opt/idea-community/bin/idea.sh" %f
Comment=IntelliJ Community Java IDE
Categories=Development;IDE;
Terminal=false
StartupWMClass=jetbrains-idea-ce
