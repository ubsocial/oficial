Habilitar Bluetooth no Debian

Habilitar complemento Bluetooth no Debian
Voltar

Alguns Desktop Environments Debian, como XFCE, não possuem o recurso de Bluetooth habilitado nativamente. Para ativá-lo, basta instalar os complementos do mesmo: sudo apt install bluez blueman
