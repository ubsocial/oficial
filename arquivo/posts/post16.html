Instalar MongoDB no Debian

Instalação do MongoDB CLI no Debian
Voltar

Passo a passo para instalação do MongoDB 5.0 no Debian 'Buster'. Para verificar a última versão do MongoDB e de seu Debian, veja as instruções abaixo.

  1. Pré-requisitos: sudo apt install gnupg wget
  2. Adicionar key: wget -qO - https://www.mongodb.org/static/pgp/server-5.0.asc | sudo apt-key add -
    • MongoDB 5.0 acima (Ver o nº da última versão no link https://pgp.mongodb.com)
  3. Adicionar repositório: echo "deb http://repo.mongodb.org/apt/debian buster/mongodb-org/5.0 main" | sudo tee /etc/apt/sources.list.d/mongodb-org-5.0.list
    • MongoDB em Debian 'buster' acima (Ver codinome da sua versão do Debian com lsb_release -sc)
  4. Atualizar lista de repositórios: sudo apt update
  5. Instalar MongoDB: sudo apt install mongodb-org
  6. Ativar e habilitar serviço: sudo systemctl enable mongod --now
  7. Testar funcionamento: mongo (Ou mongosh)
